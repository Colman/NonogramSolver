<b>Glossary</b>
<br/>
Section - 3x3 sub square on the board (indexed row by row)
<br/>
Index - Integer index of a cell. (Indexed row by row)
<br/>
Label - Another way to get a cell. Format: A0, B3, etc.
<br/>
        (Letters are columns, numbers are rows)
